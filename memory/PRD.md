# MEDVET INTEGRATIVA - PRD

## Problema Original
Site de vendas de produtos de medicina veterinária integrativa com dicas de tratamento por especialistas veterinários.

## Stack Técnica
React + FastAPI + MongoDB + Tailwind CSS + Shadcn UI + Stripe + JWT + OpenAI GPT-4o-mini

## Funcionalidades Implementadas
- **Menu duplo** (estilo MedVet.com): barra superior (ações rápidas) + barra inferior (conteúdo)
- **MEDVET News** (/noticias): 10 artigos científicos seed de journals reais, filtros, newsletter
- **Newsletter**: subscribe/unsubscribe com armazenamento MongoDB
- MEDVET Academy: 5 cursos + 20 em breve
- Portal do Veterinário: cadastro com CRMV
- Multilíngue PT/EN/ES
- Chatbot GPT-4o-mini, Meus Pets, MEDVET TV
- 25 terapias com imagens únicas (5 IA + 20 Unsplash/Pexels)
- Catálogo 77+ produtos, 9 categorias
- Checkout Stripe + PIX simulado
- Instagram + TikTok configurados
